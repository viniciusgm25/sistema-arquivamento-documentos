# Sistema de Arquivamento Digital de Documentos

Projeto acadêmico voltado à digitalização e organização de documentos institucionais, substituindo processos físicos baseados em papel.

## Objetivo
Centralizar e organizar documentos de forma digital.

## Funcionalidades previstas
- Cadastro de documentos
- Digitalização de de documentos
- Organização por CPF
- Consulta e gerenciamento de arquivos fisico e digital
- Cadastro de clientes
- Cadastro de instituições

## Tecnologias
- Java
- SQL
- Git

## Status do projeto
Em desenvolvimento.
